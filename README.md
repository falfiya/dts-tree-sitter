# dts-tree-sitter-beancount

This is a specialized version of `@asgerf/dts-tree-sitter` designed to generate `beancount.d.ts`. That's it.

## Usage

Requires:
- nodejs
- uv
- make

```sh
make beancount.d.ts
```
